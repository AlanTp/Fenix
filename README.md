Projeto Fenix

Sistema web criado para ajudar a controle de processo de um Empresa que faz Serigrafia em embalagens.

O Sistema ajuda no controle de producao, precificacao e alguns relatorios.
Tendo inicio apenas com um front estatico no vercel, o sistema foi expandindo a ponto de ter uma API, Banco Postgres e um docks no Render.

Com isso puder treiar e praticar algumas tecnologias que apredi na graducao, apromorando a experiencia como desenvolvedor.
