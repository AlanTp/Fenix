Fênix Soluções em Embalagens
Sobre o projeto

Este projeto não foi criado apenas para desenvolver um sistema web, mas principalmente para me proporcionar experiência prática em desenvolvimento Full Stack e compreender todo o ciclo de vida de uma aplicação moderna.

Durante o desenvolvimento da Fênix Soluções em Embalagens, tive a oportunidade de aplicar na prática conhecimentos adquiridos na graduação em Análise e Desenvolvimento de Sistemas, enfrentando desafios reais encontrados no mercado de trabalho.

Principais aprendizados

Ao longo do projeto pude aprender e aplicar conceitos como:

Infraestrutura e Deploy:
Como publicar uma aplicação na internet.
Como configurar ambientes de hospedagem.
Realizar deploys utilizando plataformas cloud.
Separação entre Front-end e Back-end.

Controle de Versão:
Utilização do Git para versionamento.
Criação e gerenciamento de branches.
Utilização do GitHub para armazenamento e colaboração.
Controle de evolução do código através de commits.

Desenvolvimento Front-end:
React.
JavaScript.
HTML.
CSS.
React-Bootstrap.

Desenvolvimento Back-end:
Node.js.
Express.
Criação de APIs REST.
Tratamento de requisições HTTP.
Estruturação de aplicações em camadas.

Banco de Dados:
PostgreSQL.
Modelagem de dados.
Consultas SQL.
Integração segura entre aplicação e banco de dados.

Segurança:
Autenticação de usuários.
Implementação de JWT (JSON Web Token).
Controle de acesso a rotas protegidas.

Arquitetura de Software:
Separação entre Front-end e Back-end.
Comunicação através de APIs REST.
Organização de código visando manutenção e escalabilidade.

Funcionalidades Desenvolvidas:
Controle de usuários.
Controle de produção.
Relatórios de batidas.
Controle de válvulas.
Consulta de receitas de tintas.
Emissão de pedidos.
Sistema de autenticação.
Integração com banco de dados PostgreSQL.

Objetivo
Mais do que um sistema de gestão, este projeto representa minha evolução como desenvolvedor. Foi através dele que pude transformar conhecimento teórico em experiência prática, aprendendo desde a construção da interface até a publicação da aplicação em ambiente produtivo.

Cada funcionalidade implementada representou uma oportunidade de estudar, testar soluções, corrigir erros e adquirir experiência real no desenvolvimento de software.
